### Overview
Join Bill Wolff on his 2 year journey to modernize 25+ applications using Azure cloud native techniques. We will look at Azure PIM, SQL, Aspire, Blazor, minimal API, React, wasm, app services, static web apps, storage accounts, functions, containers, service bus event based messaging, Redis caching, SignalR, app insights, log analytics, Grafana dashboards, CI/CD DevOps, Bicep, Terraform, UBA, managed identities, and open auth. There will be something for everyone!
