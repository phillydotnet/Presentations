### Overview
Vector search is a hot new feature across the Azure SQL Database platform. It can enhance real-world data scenarios with data search. In a demo-filled session, join Joey D'Antoni and learn when to use vector search, how to generate them, and the ways to integrate vectors into your data-driven app. And math!

## Joseph D'Antoni
Joseph D'Antoni brings more than 20 years of real-world experience, having worked across both Fortune 500 companies and smaller, fast-moving organizations. He combines a strong technical foundation with business insight, holding a BS in Computer Information Systems from Louisiana Tech and an MBA from NC State.

Joey is a Microsoft Data Platform MVP and a VMware vExpert. If you’ve been to PASS Summit, Ignite, SQL Saturday, or a Code Camp anywhere in the world, there’s a good chance you’ve already learned something from him. Today, he’s a Principal Cloud Architect at DesignMind, where he helps organizations turn complex data and cloud challenges into practical solutions.
